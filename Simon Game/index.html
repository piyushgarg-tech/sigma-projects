🎮 How to Play

  1. Press any key (Desktop) or tap anywhere (Mobile) to start the game.
  2. Watch the color that flashes.
  3. Click the flashing color.
  4. Each new level adds one more color to the sequence.
  5. Repeat the entire sequence correctly to continue.
  6. If you click the wrong color, the game ends.
  7. Try to reach the highest level!
